## Chức năng nổi bật

1. Đăng Nhập / Đăng Ký (Hỗ Trợ Google/Facebook).
2. Nạp Thẻ / Nạp Ngân Hàng / Nạp Ví Web Thẻ Cào (Tự Động).
3. Mua Bán Tài Khoản (Tự Động).
4. Mua Bán Dịch Vụ Game / Cày Thuê Game (Tự Động).
5. Flash Sale / Thanh Biểu Tượng Dịch Vụ / Liên Kết Nhanh ...
6. Tích Hợp Chống XSS, Lọc Bot - Request Ảo, SQL Injection.

## Lưu ý

1. Mã nguồn được mã hóa nhưng cam kết 100% SẠCH (không keylog/backdoor). Yên tâm sử dụng để kiếm tiền online!

## Các bước setup

1. Upload source, trỏ domain về thư mục gốc
2. Tạo database MySQL trống
3. Truy cập `https://ten-mien/install`
4. Bước 1: Kiểm tra hệ thống
5. Bước 2: Nhập thông tin database
6. Bước 3: Cấu hình website → **Hoàn tất cài đặt**
